# Portfólio Acadêmico - Rebeca Nalin

Este projeto é um Portfólio Acadêmico desenvolvido em Python (Flask) e utiliza SQLite para persistência de dados.

## Requisitos de Instalação

Você precisará de:

    Python 3.x (Recomendado 3.8+)
    Gerenciador de pacotes pip

### Instalação das Dependências

Abra o terminal na pasta do projeto e execute:

```bash

pip install Flask Flask-SQLAlchemy
```

⚙️ Como Rodar a Aplicação

Navegue até o diretório do Back-end:

```bash
cd backend
```
Inicialize e Rode o Servidor: O comando abaixo cria as tabelas do banco de dados (voto e contato) se o arquivo votos.db não existir e inicia o servidor. Bash

```bash
python app.py
```
Acesso: Acesse a aplicação no seu navegador: http://127.0.0.1:5000/

Pronto! Agora, você pode navegar no meu site :)
